Janee Documentation

Secure API access for AI agents — without exposing your secrets.

Janee is an open-source MCP server that acts as a secure proxy between AI agents and your APIs. Agents get scoped, auditable access to services like GitHub, Slack, and Notion — your API keys never enter the context window.
